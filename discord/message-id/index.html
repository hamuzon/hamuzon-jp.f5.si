Snowflake IDの仕組み

DiscordのSnowflake IDは64bit整数で構成されます。内容はAI生成コードによる解析ツールです。

  • 0〜41ビット: Discord Epochからのミリ秒（生成日時）
  • 42〜51ビット: Worker ID（どのサーバープロセスで生成されたか）
  • 52〜63ビット: Sequence（同じミリ秒内での連番）

Snowflake IDは単純な数字ではなく、日時順に増加する仕組みです。Binary/Hexで内部構造も確認できます。
