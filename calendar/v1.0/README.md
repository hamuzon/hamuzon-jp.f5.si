# バージョンの違い / Differences between Versions

## 📊 v1.0 の特徴 / Characteristics of v1.0

### 日本語

- 📅 **カレンダー表示**  
  日曜・土曜の色分け、今日のハイライトなど、最もシンプルな月間カレンダー表示を提供します。  
  日付のグリッドセルが画面幅に応じて均等に配置されます。

- 📝 **基本予定管理**  
  日付をクリックしてイベント（開始時刻、終了時刻、内容）を追加・削除できます。  
  この初期バージョンではまだ編集機能や詳細検索はありません。

- 💾 **データ保存とJSON入出力**  
  データはブラウザの `localStorage` に自動的に保存され、永続化されます。  
  JSON形式での保存・読み込みをサポートしており、基本的なバックアップが可能です。

---

### English

- 📅 **Calendar Display**  
  Provides the most basic monthly calendar view with Sundays/Saturdays color-coded and today's date highlighted.  
  Date grid cells are positioned uniformly relative to the screen.

- 📝 **Basic Event Management**  
  Click any date to add or delete events (start time, end time, text).  
  Note that event editing and advanced filters are not supported in this initial version.

- 💾 **Data Persistence and JSON Support**  
  Automatically saves to `localStorage` to preserve data across browser sessions.  
  Supports saving to and loading from raw JSON files for basic backups.
