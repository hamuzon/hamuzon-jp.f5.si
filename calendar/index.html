Differences in JSON Filename Format

v1.0 format

Example: calendar_events_20250710_183015.json

Example: Calendar-3.0_2025_07_10_18_30_15.json

Filenames and JSON contents differ between v1.0 and v2.0/v3.0.
However, v3.0 is backward compatible with the JSON format of v2.0. 

Beta version 4.0 includes some improvements and supports JSON formats from v1.0 to v4.0.
