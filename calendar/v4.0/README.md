# バージョンの違い / Differences between Versions

## 📊 v3.0 と v4.0 の違い / Differences between v3.0 and v4.0

### 日本語

- 📅 **ICS (iCalendar) のインポート・エクスポート対応**  
  v3.0 ままではJSON形式のみでしたが、v4.0 では業界標準のICS形式に対応しました。  
  OutlookやGoogleカレンダーなど他のカレンダーサービスと相互に予定データを移行可能です。  
  また、ワンクリックの保存形式を設定画面からJSONかICSかを選択して固定できるようになりました。

- 🎨 **UIデザインのプレミアム化とダークモード自動判定**  
  滑らかなアニメーション、より直感的でスタイリッシュな角丸やシャドウが追加されました。  
  テーマ選択（ライト・ダーク・OS連動の自動）に対応した高品質なCSSシステムが導入されました。

---

### English

- 📅 **ICS (iCalendar) Import & Export**  
  While v3.0 only supported JSON, v4.0 supports the industry-standard ICS format.  
  Users can now import and export event data to sync with Google Calendar, Outlook, and Apple Calendar.  
  You can also set and lock your preferred one-click save format (JSON or ICS) from settings.

- 🎨 **Premium UI Styling and Automatic Theme Engine**  
  Introduces modernized glassmorphic shadows, smooth transitions, and a premium CSS theme engine.  
  Fully supports Light, Dark, and Auto (OS-linked) theme selections.
